🏭 CEMPACKSYS – Cement Packing Monitoring System
📌 Overview

CEMPACKSYS is a web-based Manufacturing Information System (MIS) developed using ASP.NET Core MVC to monitor cement packing machine operations.

The system collects and visualizes packer and spout weight data, helping plant operators track production accuracy, tonnage, and machine performance in real time.

It provides both data tables and analytical dashboards to support operational monitoring and decision-making.

🎯 Key Features

📊 Production Data Monitoring

Displays real-time packing data including:

• Packer Number

• Spout Number

• Bag Weight

• Timestamp

• Production Records

This allows operators to monitor packing machine outputs continuously.

📈 Weight Analytics Dashboard

The dashboard provides key production insights such as:

• Total Tons Produced

• Average Bag Weight

• Total Production Records

• These metrics help track overall production performance.

⏱ Tons Discharged Per Hour

Visualizes hourly production output to monitor plant throughput and identify production trends.

⚖ Average Weight Per Spout

Displays weight distribution across multiple spouts of the packing machine to detect inconsistencies between spouts.

📉 Production Trend Visualization

• Interactive graphs provide insights into:

• Weight fluctuations

• Production rate changes

• Spout performance comparison

🔍 Date Range Filtering

Users can select custom date and time ranges to analyze production data within specific operational periods.

🖥 System Modules

1️⃣ Data Monitoring Module

• Displays live production data in a structured table for quick inspection.

2️⃣ Analytics Dashboard

• Provides summarized production metrics and graphical insights.

3️⃣ Reporting Module

• Generates visual reports for analyzing packing machine performance.

🛠 Technologies Used

📊 Key Operational Insights

The system helps plant engineers and supervisors:
